
# Flask application


1. Create a virtual environment:
    
```
python -m venv venv
```


2. Activate the virtualenv:

Linux:
```
source venv/bin/activate
```

Windows:
```
venv\Scripts\activate
```

3. Install flask, and python-dotenv:

```
pip install flask python-dotenv yapf

```

4. Run the application with:

```
flask run
```
